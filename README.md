# URL Shortener Web Application

This is a URL Shortener Web Application developed using Python and Flask.

## Features
- Shortens long URLs
- Redirects short URLs to original URLs
- Copy short URL with one click
- Uses SQLite database

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite

## How to Run
1. Install Flask
2. Run python app.py
3. Open browser and go to http://127.0.0.1:5000
